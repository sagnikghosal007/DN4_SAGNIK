# Exercise 4: Employee Management System

##  Features
- Add employee
- Search by employee ID
- Traverse/display all employees
- Delete by employee ID

##  Time Complexity
| Operation | Complexity |
|-----------|------------|
| Add       | O(1)       |
| Search    | O(n)       |
| Traverse  | O(n)       |
| Delete    | O(n)       |
